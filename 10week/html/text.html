문제 1번

1. 동적 웹페이지와 정적 웹페이지의 차이를 설명하시오.
동적 웹페이지와 정적 웹페이지는 웹 페이지의 콘텐츠가 어떤 방식으로 생성되어 화면에 나타나는지로 구분할 수 있습니다.
각각의 특징은 아래와 같습니다.
[정적 웹페이지(Static Web Pages)]
- 서버에서 클라이언트 측으로 전송되기 전 이미 HTML, CSS, JavaScript 등의 파일로 구성되어 미리 만들어진 콘텐츠를 가지고 있음
- 사용자가 페이지를 요청할 때마다 항상 동일한 내용이 표시
- 변경이 필요한 경우에는 새로운 HTML 파일을 생성하여 업데이트해야 합니다.
=> 주로 정보가 자주 변경되지 않는 웹사이트에 적합.

[동적 웹페이지(Dynamic Web Pages)]
- 동적 웹페이지는 사용자의 요청에 따라 콘텐츠가 생성되고 표시됩니다.
- 서버는 사용자의 요청을 받으면 그에 맞게 콘텐츠를 동적으로 생성하여 전송합니다.
- 데이터베이스와 같은 백엔드 기술을 사용하여 실시간으로 정보를 업데이트하고 사용자에게 동적인 경험을 제공합니다.
- 예를 들어, 온라인 상점의 장바구니 업데이트, 실시간 검색 결과, 사용자 인증 등이 동적 웹페이지에서 이루어집니다.
동적 웹페이지는 사용자와의 상호작용이 많은 웹 애플리케이션에 적합하며,
주로 웹 프레임워크(예: Django, Flask, Ruby on Rails)를 사용하여 개발됩니다.
반면에 정적 웹페이지는 주로 간단한 정보 제공용으로 사용되며, 서버 부하가 적고 속도가 빠르다는 장점이 있습니다.
